# Picture Perfect

Een website waar je foto's kan posten om deze met andere mensen te sharen.

# Wat kan je doen op de website
Eens je ingelogd bent kan je foto's zien van andere gebruikers en natuurlijk kan je deze ook een like geven als je het een mooie, creatieve of originele foto vindt.
Ben je benieuwd naar welke foto's je vrienden hebben gepost? Dan zoek je hun gewoon op en dan kun je naar hun profiel gaan waar je enkel hun foto's ziet.
Dit is ook mogelijk voor je eigen profiel natuurlijk!
Per post (die meerdere foto's kan bevatten) kun je de beschrijving, die de gebruiker zelf heeft geschreven, en de datum van wanneer deze geplaatst is zien.

## How to
Om de applicatie te runnen moet het commando "npm start" uitgevoerd worden in de CLI van het project alsook moet de API draaien op localhost:5001 en is swagger te vinden op localhost:5001/swagger/index.html. Voor de authenticatie te testen in swagger moet de definitie rechtsboven veranderd worden naar apidocs.
